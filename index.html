🪣
Amazon S3
Scalable object storage for raw, processed, and query-result data. Supports Bucket Policy (JSON) and Bucket ACL (XML) access controls.
🕷️
AWS Glue Crawler
Automatically discovers and catalogues schema from S3 data sources, populating the Data Catalog without manual intervention.
📋
Glue Data Catalog
Central metadata repository that makes datasets queryable via Athena. Acts as the bridge between storage and analytics.
🔍
Amazon Athena
Serverless interactive query service that analyses data directly in S3 using standard SQL. Query results are saved back to S3.
📊
Amazon QuickSight
Fully managed BI service that connects to Athena to render interactive charts, candlestick graphs and EMA comparison dashboards.
⚡
Amazon EC2
Compute instances used to run data ingestion scripts (Python), shell scripts, and CSV generation before uploading to S3.
02 — Data Processing

ETL with AWS Glue

Data Lake Architecture
🥉
Bronze Layer
Raw ingestion and historical storage. All original data lands here untouched — the immutable source of truth from Kafka, Kinesis, CSV/JSON/TXT files and Spark streams.
Kafka Kinesis CSV / JSON Raw
🥈
Silver Layer
Filtered, cleaned and augmented data. Glue ETL jobs transform raw records into validated, deduplicated datasets ready for analytics consumption.
Filtered Cleaned Augmented Validated
🥇
Gold Layer
Business-level aggregates optimised for BI reporting, streaming analytics and ML model training. KPIs, EMAs, SMAs and candlestick data live here.
BI & Reporting ML Ready Aggregated KPIs
